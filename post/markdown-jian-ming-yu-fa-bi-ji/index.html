图片

img标签

src,图片链接，图片地址
width height高度宽度
alt备选内容，当图片内容丢失时显示的值
href跳转链接地址

<img src="xxxxx.jpg" width="80px" alt="图片丢失">

视频

video标签
src,视频链接，视频地址
width height高度宽度
controls视频控制控件
autoplay自动播放
preload预载视频，有none，auto
loop循环播放
poster视频载入时显示封面图片
muted视频静音

<vedio src="xxxxx.mp4" height="800px" controls preload="auto" poster="xxx.img"></vedio>
