为Gridea主题添加看板娘

  1. 2020-04-06
  2. 321字
  3. 2 min read

Add Live2D for Gridea theme

概要

看到别人的网站有看板娘，但是Gridea并没有特别完整的解决方案，便在Github上寻找相关的代码，看到项目：live2d-widget，便稍作修改，就实现将看板娘本地化到主题中，需要改动的地方很少。
效果预览：wherelse.cc

操作方法

  1. 从我的github上下载或clone repo，地址：Live2D For Gridea

  1. 将文件以文件夹形式放入到\主题\assets\media下


    文件夹的内容应有一下几项：

  2. 将下面的代码加入 index.ejs的<head>或 <body>之间保存。

<script src="./media/live2d/autoload.js"></script>

在Gridea中预览添加完成后的效果。

在主题中添加开启按钮（可选）

在config.json最后添加：

{
    "name": "Live2D",
    "label": "Live2D看板娘",
    "group": "Live2D",
    "value": true,
    "type": "switch",
    "note": "若不需要，可关闭以加快页面加载速度"
}

<head>或 <body>间的代码更改为：

<% if (site.customConfig.Live2D){ %>
    <script src="./media/live2d/autoload.js"></script>
<% } %>

Live2D自定义

在waifu-tips.json和waifu-tips.js中可以自定义看板娘的文字内容，按需更改。
